Document Sign-Off Example (Tier 0 - Task)

This example demonstrates the simplest possible process in the process-as-universal architecture: a Tier 0 (Task) that tracks status and outcome without any stages or obligations. The document-signoff universal specializes base-process and adds two custom properties.

Overview

Capability Tier: 0 (Task)
Composition Depth: 0 (no stages composed in)
Key Feature: Inherits all base-process properties (status, timestamps, outcome) via generalization

Key Features Demonstrated

  • Generalization: document-signoff inherits all base-process properties
  • Custom Properties: Adds domain-specific properties (document_ref, reviewer_notes)
  • No Composition: No stages or obligations — simplest possible process
  • Tier Detection: Engine inspects composition tree, finds depth 0 = Task tier
