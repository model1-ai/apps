Content Moderation Example (Tier 1 - Worklist)

This example demonstrates a Tier 1 (Worklist) process with 3 sequential stages composed into the process universal via composition-relator. When a moderation process instance is created, composition auto-instantiation automatically creates the 3 stage instances.

Overview

Capability Tier: 1 (Worklist)
Composition Depth: 1 (process → stages)
Key Feature: Stages auto-created via composition auto-instantiation (Phase 1)

Key Features Demonstrated

  • Stage Composition: 3 stages composed into process via composition-relator
  • Auto-Instantiation: Stage instances created automatically on process creation
  • Sequential Stages: Stages activate in sequence order
  • No Obligations: Stages are worked from a worklist/kanban view without role assignments
  • Tier Detection: Engine finds composition depth 1 (stages only) = Worklist tier
